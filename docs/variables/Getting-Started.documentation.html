Getting Started with FARM

FARM (Full Stack • API-first • Reactive • Modern Web Platform) is a modern web application boilerplate that combines the best practices of full-stack development.

Prerequisites

  • Node.js >= 22.0.0
  • npm >= 10.0.0

Quick Start

  1. Clone the repository:
git clone https://github.com/Falkicon/farm.git
cd farm
  1. Install dependencies:
npm install
  1. Set up environment:
cp .env.example .env.local
  1. Start development server:
npm run dev

Project Structure

  • /src/frontend - Web Components and UI logic
  • /src/backend - Fastify server and API routes
  • /src/system - System metrics and monitoring
  • /src/shared - Shared utilities and types
  • /src/tests - Test suites and utilities

Key Features

Frontend

  • Lit Web Components
  • TypeScript
  • Tailwind CSS
  • Hot Module Replacement
  • Storybook Integration

Backend

  • Fastify Server
  • API-First Development
  • Prisma Database Integration
  • System Metrics

Development

  • TypeScript Throughout
  • Comprehensive Testing (Vitest & Playwright)
  • TypeDoc Documentation
  • GitHub Actions CI/CD

Available Scripts

  • npm run dev - Start development server
  • npm test - Run tests
  • npm run build - Build for production
  • npm run docs - Generate documentation
  • npm run storybook - Start Storybook
