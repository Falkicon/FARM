Development Overview¶

Introduction¶

This section covers the development practices, guidelines, and processes for the FARM Stack project. Our development approach focuses on maintainability, type safety, and developer experience.

Quick Links¶

  • Development Guidelines - Coding standards and best practices
  • Testing Guide - Testing strategies and practices

Development Environment¶

Prerequisites¶

  • Node.js 22.0 or higher
  • npm 10.0 or higher
  • Git
  • Python 3.x (for documentation)
  • Git Bash (recommended) or another terminal

Getting Started¶

  1. Clone the repository
  2. Install dependencies
  3. Set up your development environment
  4. Start the development servers

For detailed setup instructions, see the Installation Guide.

Development Workflow¶

1. Code Quality¶

# Run linting
npm run lint

# Type checking
npm run typecheck

# Run tests
npm run test

2. Development Servers¶

# Start all development servers
npm run dev

# Start individual servers
npm run dev:frontend    # Frontend on port 3000
npm run dev:backend     # Backend on port 8000
npm run storybook       # Storybook on port 6006

3. Documentation¶

# Start documentation server
npm run docs

# Build documentation
npm run docs build
