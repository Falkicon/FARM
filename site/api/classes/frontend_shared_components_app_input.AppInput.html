Class AppInput

Input field component with label and change event handling

Remarks

Provides a styled input field with optional label and placeholder. Uses Tailwind CSS classes for consistent styling and focus states.

Example

<app-input
  label="Username"
  placeholder="Enter username"
  @change=${(e) => console.log(e.detail)}
></app-input>
